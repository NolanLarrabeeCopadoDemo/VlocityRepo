Release Notes for Customer Insights

Version 1.1

New Dashboards

Sales Performance

  • Ready-to-deploy dashboard with the most commonly requested sales metrics to track revenue, pipeline, leads, activities, and top accounts.
  • Minimal use of SAQL and Interactions (Bindings) for easy customization.
  • Demonstrates design and usability best practices.

Opportunity Embedded

  • Designed to embed in the Opportunity page layout using Lightning App Builder.
  • Use the Filter Builder to add filters for Activity, Case, Opportunity, and OpportunityHistory
    • Activity Dataset: Match Oppd.Id to Id
    • Case Dataset: Match AccountId to AccountId
    • Opportunity Dataset: Match Id to Id
    • OpportunityHistory Dataset: Match OpportunityId to Id

Modular Recipes

Customer Insights can now run in Orgs without role hierarchy, lead records, or case records. Instead, Salesforce loads sample data into the Sales Performance Dashboard so you can see a preview of all widgets

Row-Level Security

Security Predicates are now added by default on the following datasets

  • Activity
  • Opportunity
  • OpportunityHistory
  • OpptyProduct
  • Account
  • Lead

Users will be able to view data owned by them and others in their role and subordinates. In Orgs without role hierarchy, users will be able to view data owned by them
